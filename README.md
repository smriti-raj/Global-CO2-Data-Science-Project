 🌍 Global CO₂ Emissions Analysis
 
📌 Project Overview
This project analyzes historical global CO₂ emissions using a real-world environmental dataset containing 79 variables related to carbon emissions, greenhouse gases, GDP, population, and energy consumption.

The objective is to identify emission trends, compare countries, explore relationships between economic indicators and emissions, and build predictive machine learning models.

📂 Dataset
Dataset: Global CO₂ Database
Rows: Thousands of observations
Columns: 79

Main Features
- Country
- Year
- Population
- GDP
- CO₂ Emissions
- CO₂ Per Capita
- Energy Consumption
- Methane
- Nitrous Oxide
- Total Greenhouse Gas Emissions
- Trade CO₂
- Temperature Change
  
🎯 Objectives
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Missing Value Analysis
- Correlation Analysis
- Country-wise CO₂ Comparison
- Time Series Trend Analysis
- Machine Learning Prediction
- Business & Environmental Insights

  📈 Key Insights

- Global CO₂ emissions have increased significantly over time.
- Countries with larger populations generally contribute more to total emissions.
- Higher GDP is often associated with higher CO₂ emissions, although the relationship varies across countries.
- CO₂ per capita differs substantially between countries, highlighting differences in industrialization and energy use.
- Correlation analysis identifies the strongest relationships among economic, demographic, and environmental variables.
